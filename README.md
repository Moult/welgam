# Welgam

Welgam is a Weight Loss Gamification programme for people wanting to lose (or
gain, actually) weight.

## Usecases supported

Those marked in bold have been implemented:

 * **Competition add**
 * **Competition edit**
 * **Competition delete**
 * **Racer add**
 * **Racer edit**
 * **Racer delete**
 * Racer award
 * Update add
 * Update delete
